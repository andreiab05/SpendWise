# SpendWise

SpendWise is a desktop application for managing monthly budgets throughout the year.

The app allows users to create budget entries for different spending categories, track how much money has been spent, and compare it with the allocated monthly budget. Past years are read-only, so older budget data can be viewed but not modified.

I built this app because I wanted a simple way to track spending without relying on paper notes or overly complicated budgeting apps.

## Features
- Add monthly budget entries
- Delete monthly budget entries
- Filter entries by year and month
- Add a category to all months of a selected year
- Increase spent amount for an existing category
- Highlight over-budget entries
- Prevent editing past years

## Tech Stack
- Java
- JavaFX
- SQLite
- Maven
- Layered architecture
